# OAuth Setup Guide

## OAuth Configuration Steps

1. Create an application in the developer portal.
2. Generate Client ID and Client Secret.
3. Configure authorized redirect URLs.
4. Enable required API scopes.
5. Save configuration settings.

---

## Common OAuth Errors

### Invalid Redirect URI

Cause:
The redirect URI does not match the registered URI.

Resolution:
Update the application settings to match the correct redirect URI.

---

### Invalid Client Credentials

Cause:
Incorrect Client ID or Client Secret.

Resolution:
Regenerate credentials and update the application configuration.
